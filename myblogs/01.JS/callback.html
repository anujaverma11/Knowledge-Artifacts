Javascript Callback Functions

12/14/2015

A callback function is a higher-order function, that is passed to another function as a parameter. And then this function is called inside the other function. Here in the below code greeting is a callback function which is passed to closeTerminal as a parameter and later called inside it. 

var greeting = function() {
            alert("Thanks for visiting");
            };
            closeTerminal(greeting);
            function closeTerminal(message)
            {
            message();
            }

Function Callback with Array and Map

Example 1

var passengers = [ ["Thomas", "Meeks"],
                   ["Gregg", "Pollack"],
                   ["Christine", "Wong"],
                   ["Dan", "McGaw"] ];
                  var modifiedNames = passengers.map(function(arrayCell){
