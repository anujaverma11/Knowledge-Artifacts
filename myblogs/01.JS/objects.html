Javascript Objects

12/16/2015

Objects are multiple pieces of Data, called properties, are grouped within an object.

Creating an Object with Javascript

There are multiple ways to create an object using object Literals and using object.create().

1. Object Literal

A set of curly brackets says to make a new object. As shown below myBox is an empty object

var myBox = {};

To add a property invloves creating a name for the property unsing a string and then assigning a value to it using a ':' (colon)

var booksArray = ["Great Expectations","The Remains of the Day","Peter Pan"]
var myBox = {height: 6, width: 8, length: 10, volume: 480,
             material: "cardboard",
             contents: booksArray
           };

Access Property values

To access any property of this obeject, simply use '.' with object name

for example:

myBox.height;

myBox.width;

or

myBox["volume"]

Changing property values

The dot operator also allows modification of properties, even using methods.

myBox.width = 12;

or

myBox["width"] = 12

will change the width value from 8 to 12.

Using Brackets also help in accessing property name saved as string with spaces.

myBox["# of stops"] = 2;

This cannot be done with Dot operator.

Deleting a Property

delete myBox.contents;

This will delete the entire property with its value. This however will not delete the booksArray. If we console.log the booksArray we will still get the Array, as this deletes the property name and the refernce, bit not the original booksArray outside the Box.

Generic objects don't have a length like Arrays and strings do, so we cannot use that in a loop format in order to get to each property.

For-in Loop

The for-in loop allows us to access each enumerable property in turn.

An enumeration is a complete, ordered listing of all the items in a collection.

for(key in lighthouseRock) {

console.log(key)

}

key here can be any keyword.

In the below example, listguns is a function which belong to rockSpearguns, hence there is no need to pass any parameter to this function and "var property in this" will check each property of rockSpearguns object. 

var rockSpearguns = {

// convert listGuns function
rockSpearguns[listGuns] = function() {
  for (var property in this) {
    if(this[property]["heft"]!=undefined)

}

Examples of Objects

Example 1

var vehicle1 = {type: "Motorboat", capacity: 6, storedAt: "Ammunition Depot"};
                              var vehicle2 = {type: "Jet Ski", capacity: 1, storedAt: "Reef Dock"};
                              var vehicle3 = {type: "Submarine", capacity: 8, storedAt: "Underwater Outpost"};

                              // create vehicles array

                              var vehicles = [vehicle1,vehicle2,vehicle3];

                              // build findVehicle function expression
                              var findVehicle = function(vehicleName, list){
                                for(var i=0; i<=list.length;i++)
                                  {
                                    if (vehicleName == list[i].type)
                                      {
                                        return list[i].storedAt;
                                      }
                                  }
                              };

                              // call findVehicle

                              findVehicle("Submarine", vehicles);

Example 2

var superBlinders = [ ["Firestorm", 4000], ["Solar Death Ray", 6000], ["Supernova", 12000] ];

var lighthouseRock = {
  gateClosed: true,
  weaponBulbs: superBlinders,
  capacity: 30,
  secretPassageTo: "Underwater Outpost",
  numRangers: 0
};

function addRanger(location, name, skillz, station) {
  // increment the number of rangers property
  location.numRangers++;
  // add the ranger<number> property and assign a ranger object
  location["ranger"+location.numRangers] = {name: name, skillz: skillz, station: station};

}

// call addRanger three times to add the new rangers
addRanger(lighthouseRock,"Nick Walsh","magnification burn", 2);
addRanger(lighthouseRock,"Drew Barontini","uppercut launch",3);
addRanger(lighthouseRock,"Christine Wong","bomb defusing", 1);
