Understanding Hex Colors

#FFFF00

Here the first two digits 'FF' determine the amount of red, next two determine the amount of green and last two digits determine the amount of blue.

Hexadecimal numbers run from 0-15, but use letters for numbers above 9.

  • A = 10
  • B = 11
  • C = 12
  • D = 13
  • E = 14
  • F = 15

Block-Level Tags - The content of block-level tags take up the entire width of the container.

Inline-Level tags - They do not take all the width of the container but just flow with the content.

The Box Model

The Box Model is a way to describe the borders and spacing in between the boxes of each tag.

There are 4 parts of the box Model:

  • Content Area
    • The content area contains the actual content(text, image, etc)
  • Padding
    • Padding is added to the top, right, bottom or left of the content area.
  • Border
    • Borders are added around the top, right, bottom or left of the padding.
  • Margins
    • Margins are added to the top, right, bottom or left of the border.

The full size of a box after these four properties have been set can be calculated as shown below:

the full box width = content area width + padding-left + padding-right + border-left + border-right + margin-left + margin-right

Padding Setting

We can adjust padding doing the following style settings: 
h2
          {
          padding-top: 6px;
          padding-right: 6px;
          padding-bottom: 6px;
          padding-left: 6px;
          }
or they can be done in a single line as follows: 
h2
        {
        padding: 6px 6px 6px 6px;
        }

Padding is used to control the size of a box without adjusting the size of the content inside the box.

Border Setting

h2
        {
        border-width: 6px;
        border-style: solid;
        border-color: black;
        }

To add border just one side, use border-top, border-right, border-bottom, border-left

Margin Setting

h2 {
          margin-top: 6px;
          margin-right: 6px;
          margin-bottom: 6px;
          margin-left: 6px;
        }

Margin is used to control the space between boxes.

Quick Notes

  1. To get space between the lines of a paragraph adjust line-height property.

    2. p {
          color: #444444;
          margin: 0 0 10px 0;
          line-height: 24px;
          }

  2. To select a tag with its attribute:

    3. <form>
            <label for="band-name">Band Name</label>
            <input type="text" id="band-name">
            <label for="your-story">Your Story</label>
            <textarea id="your-story"></textarea>
            <label for="on-tour">On Tour</label>
            <input type="checkbox" id="on-tour">
            <input type="submit" value="Tell us about your band">
            </form>
    use the following CSS: 

    input[type=submit]{
            width: 165px;
            }
            input[type=checkbox]{
            display: inline;
            }
            label[for=on-tour] {
            display: inline;
            }
