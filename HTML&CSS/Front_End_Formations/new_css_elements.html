Border Radius - rounded corner to box elements.
Box Shadow - gives a shadow as specified. box-shadow:<inset> <offset-x> <offset-y> <blur-radius> <spread-radius> <color> box-shadow: 1px 2px 2px #000;
inset - default is drop shadow; if this is specified then an inset shadow.
offset-x - The shadow offset x value (left or right)
offset-y - The shadow offset x value (up and down)
blur-radius - how blur you want
spread-radius - how much shadow spread you want
color - color of the shadow
Text Shadow - applies the shadow to text text-shadow: 1px 2px 2px #000;
